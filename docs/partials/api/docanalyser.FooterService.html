FooterService
service in module docanalyser

Description

The service is responsible for displaying alerts in the lower footer area.

Methods

  • hideWarning()

    close the custom message in the footer area of the website.

    Example

    FooterService.hideWarning();

  • showWarning(message)

    show a message in the footer area of the website.

    Parameters
    ParamTypeDetails
    messageString

    message to be displayed

    Example

    FooterService.showWarning("custom Message");

Properties

  • warningDiv:Object

    Global object for the warning message.

  • showWarning:Object

    Global object indicating whether to display a warning.
