PastQueriesMenuCtrl
controller in module docanalyser

Description

Controller for the management of the saved searches.

Methods

  • clearHistory()

    user clicked the Clear History button. All stored queries are deleted.

    Returns
    boolean

    true if all queries could be deleted

    Example

    clearHistory();

  • init()

    Initialization stuff. In case of page reload, stored queries are reloaded again. The chronicle status is loaded or set to true at the first page view.

    Example

    init();

  • loadQuery(query)

    Load a query: set loaded keywords, topics, top 4 keywords and perform a google search

    Parameters
    ParamTypeDetails
    queryObject

    Query to be loaded

    Example

    loadQuery(mySelectedQuery);

Properties

  • chronicleStatus:boolean

    Specifies whether the user wants to save queries or not.

  • chronicleStatus:Array

    Contains the stored queries.

  • searchText:String

    Contains the filter input.
