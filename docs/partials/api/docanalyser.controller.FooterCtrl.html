FooterCtrl
controller in module docanalyser

Description

Controller responsible for displaying alerts and other information at the bottom of the webpage.

Methods

  • hasAgreed()

    Check if the user has consented to the use of cookies.

    Returns
    boolean

    true if yes

  • setCookieValue()

    The user has consented to the use of cookies. An appropriate cookie is set for the page to remember this decision.

Properties

  • showWarning:boolean

    Specifies whether to display a warning.

  • warningDiv:String

    Content of the warning
