KeywordsMenuCtrl
controller in module docanalyser

Description

Controller for the management of the keywords. Here user interactions are treated.

Methods

  • clicked()

    The user has checked a keywords checkbox. Informs the SearchInputCtrl that a term has been selected.

  • init()

    Initialization stuff. Synchronizes the status of keywords and source topics.

  • performSearch()

    Synchronizes the status of keywords and source topics and perform a google search.

  • queryLoaded()

    A query is loaded (from the Chronicle View). The top four keywords are written in the search bar and a google search is performed.

    Example

    $rootScope.$broadcast('queryLoaded');

Properties

  • keywords:Object

    Contains all keywords.
