resizable
directive in module docanalyser

Description

This directive causes the right-hand pane to adjust its width when resize events occur. "resize" events occur when the user changes the width of the left menu or the browser window size changes.

Usage

as attribute
<ANY resizable>
   ...
</ANY>
as class
<ANY class="resizable">
   ...
</ANY>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 1000.
